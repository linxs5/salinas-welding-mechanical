## Deployment

This site uses plain HTML, CSS, and minimal JavaScript. There is no framework, no build step, and no package install required.

For Netlify Free deployment from GitHub:

1. Push this folder to a GitHub repository.
2. In Netlify, choose **Add new site** and connect the repository.
3. Leave the build command blank.
4. Set the publish directory to the repository root.
5. Deploy the site.
6. After deployment, open Netlify Forms and confirm the `quote` form appears.
7. Add the notification email for quote requests.
8. Submit a test quote request before launch.
